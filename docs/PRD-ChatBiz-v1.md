## 一、产品概述

### 1.1 产品定位

ChatBiz（洽聊）是一款面向跨境电商和外贸企业的多平台智能客服管理系统。产品整合 11 个主流社交/电商平台的客户会话，通过 AI 辅助销售、客户画像、知识库等能力，帮助企业实现统一的客户沟通管理和销售转化提升。

### 1.2 核心价值

| 价值维度 | 描述 |
|----------|------|
| 多平台统一管理 | 一个后台管理 WhatsApp、Telegram、Instagram 等 11 个平台的客户消息 |
| AI 驱动销售 | AI 员工自动接待、智能推荐、主动营销，降低人力成本 |
| 客户资产沉淀 | AI 画像标签体系 + 知识库，将客户数据转化为企业资产 |
| 内控合规 | 聊天记录审计、敏感词监控、操作报表，满足企业管理需求 |

### 1.3 目标用户

| 角色 | 场景 | 核心诉求 |
|------|------|----------|
| 外贸企业主/管理者 | 管理多个销售账号，监控团队工作 | 数据可视化、内控合规、成本控制 |
| 跨境电商运营 | 多平台客户沟通、营销触达 | 统一收发消息、AI 辅助回复、客户画像 |
| 客服主管 | 分配工单、监控服务质量 | 工单管理、质量检测、效率报表 |
| 一线客服/销售 | 日常客户沟通 | 快速回复、翻译、AI 建议、话术库 |

### 1.4 支持平台（11 个）

| 平台 | 类型 | 主要市场 |
|------|------|----------|
| WhatsApp | 即时通讯 | 全球（拉美、中东、东南亚） |
| Telegram | 即时通讯 | 全球（俄罗斯、中东） |
| Line | 即时通讯 | 日本、泰国、台湾 |
| Instagram | 社交媒体 | 全球 |
| Facebook | 社交媒体 | 全球 |
| WeChat（微信） | 即时通讯 | 中国 |
| Email | 电子邮件 | 全球 |
| SMS | 短信 | 全球 |
| TikTok | 短视频/直播 | 全球 |
| Twitter | 社交媒体 | 全球 |
| Shopify | 电商平台 | 全球 |

## 三、功能模块 — 客服工作台

### 3.1 多平台会话管理

会话列表支持按平台、状态、优先级、标签等多维度筛选，支持单聊和群聊两种会话类型。

| 功能项 | 说明 |
|--------|------|
| 会话列表 | 按最新消息排序，显示未读数、平台图标、客户头像、最后一条消息预览 |
| 聊天窗口 | 消息气泡区分客户/客服/AI 三种发送者，AI 生成消息有标记 |
| 消息状态 | sent → delivered → read，实时更新 |
| 群聊支持 | 显示群名称、群成员数，群消息聚合展示 |
| 会话状态 | active（进行中）/ pending（待处理）/ resolved（已解决）/ closed（已关闭） |
| 优先级 | low / medium / high / urgent，支持手动调整 |

筛选条件：

| 筛选维度 | 选项 |
|----------|------|
| 平台 | 11 个平台多选 |
| 状态 | active / pending / resolved / closed |
| 优先级 | low / medium / high / urgent |
| 聊天类型 | 全部 / 单聊 / 群聊 |
| 未读/未回复 | 仅未读 / 仅未回复 |
| 国家/语言 | 多选 |
| 标签 | 多选 |
| 关键词搜索 | 模糊匹配客户名称、消息内容 |

### 3.2 多平台账号管理

每个平台支持添加多个社交账号，统一在左侧边栏管理。

| 功能项 | 说明 |
|--------|------|
| 账号添加 | 按平台添加社交账号，填写账号 ID、昵称、手机号等 |
| 账号状态 | online / offline / busy / not_logged_in |
| 代理 IP 绑定 | 每个账号可绑定独立代理 IP，支持 HTTP/HTTPS/SOCKS5 协议 |
| AI 开关 | 单账号级别开启/关闭 AI 自动回复 |
| 默认账号 | 每个平台可设置一个默认账号 |

代理 IP 配置：

| 字段 | 说明 |
|------|------|
| 名称 | 代理配置名称 |
| 协议 | HTTP / HTTPS / SOCKS5 |
| 主机 / 端口 | 代理服务器地址 |
| 用户名 / 密码 | 认证信息（可选） |
| 地区 | 代理所在地区 |

### 3.3 AI 辅助工具（会话级）

在聊天窗口中，客服可使用以下 AI 辅助功能：

| 功能 | 说明 |
|------|------|
| AI 回复建议 | 每条会话自动生成 3 条回复建议，一键采用 |
| AI 自动翻译 | 接收消息自动翻译为中文，发送消息自动翻译为客户语言 |
| AI 会话总结 | 自动生成当前会话的摘要，快速了解上下文 |
| AI 接管模式 | 开启后 AI 全自动回复客户消息 |

### 3.4 右侧面板

聊天窗口右侧可展开多个辅助面板：

| 面板 | 内容 |
|------|------|
| AI 客户画像 | 展示客户的 AI 标签（等级、类型、意向品类、预算等） |
| 代理 IP 配置 | 当前账号的代理设置 |
| 翻译面板 | 翻译偏好设置、语言切换 |
| 联系人详情 | 客户基本信息、订单历史、行为分析 |

### 3.5 仪表盘（Dashboard）

客服工作台首页，展示当日工作概览：

| 指标 | 说明 |
|------|------|
| 待处理会话数 | 当前 pending 状态的会话 |
| 未读消息数 | 所有会话未读消息总计 |
| 今日接待客户数 | 当日有消息往来的客户数 |
| 平台分布 | 各平台会话数量占比 |

### 3.6 设置

| 设置项 | 说明 |
|--------|------|
| 语言偏好 | 界面语言（10 种语言） |
| 时区 | 用户所在时区 |
| 通知设置 | 新消息通知开关 |
| 翻译偏好 | 自动翻译开关、目标语言 |
| AI 偏好 | AI 回复建议开关、自动总结开关 |

## 四、功能模块 — 管理中心

### 4.1 激活码管理

激活码是系统的核心准入机制，每个客服/销售人员通过激活码获得系统使用权限。

| 功能项 | 说明 |
|--------|------|
| 部门树形结构 | 左侧展示组织部门层级（如：销售部 → 国内销售组 / 海外销售组） |
| 激活码列表 | 按部门筛选，展示所有激活码及其状态 |
| 创建激活码 | 指定部门、角色、有效期、备注 |
| 平台绑定 | 每个激活码可绑定允许使用的平台 |
| AI 坐席配置 | 设置 AI 坐席上限、分配模式（固定/自动） |

激活码状态流转：

```
unused（未使用）→ active（已激活）→ expired（已过期）
                                  → disabled（已禁用）
```

激活码字段：

| 字段 | 说明 |
|------|------|
| 编号 | 系统生成的唯一激活码 |
| 所属部门 | 关联的组织部门 |
| 分配人员 | 使用该激活码的人员姓名 |
| 角色 | admin / manager / agent |
| 状态 | unused / active / expired / disabled |
| AI 坐席上限 | 该激活码可使用的 AI 坐席数量 |
| AI 分配模式 | fixed（固定分配指定账号）/ auto（自动分配） |
| 创建时间 / 激活时间 / 到期时间 | 时间戳 |

### 4.2 客户管理

企业级客户数据中心，汇聚所有平台的客户信息。

#### 4.2.1 客户列表

两行筛选栏 + 表格列表 + AI 画像标签侧边栏筛选。

第一行筛选：

| 筛选项 | 类型 |
|--------|------|
| 激活码 | 下拉选择 |
| 社交平台 | 下拉选择（11 平台） |
| 国家 | 下拉选择 |
| 社交账号昵称 | 文本输入 |
| 客户状态 | 下拉选择（正常/取消跟进/已删除） |

第二行筛选：

| 筛选项 | 类型 |
|--------|------|
| 活跃度 | 下拉选择（高/中/低） |
| 昵称备注 | 文本输入 |
| 线索录入时间 | 日期范围 |
| 记录时间 | 日期范围 |

活跃度计算规则：
- 高：3 天内有联系
- 中：3-14 天内有联系
- 低：超过 14 天无联系

AI 画像标签筛选（侧边栏）：
- 展示三级字段 + 四级可选值
- 字段间 AND 关系，字段内 OR 关系
- 支持按行业分组切换

操作功能：批量选择、导出数据、添加客户

#### 4.2.2 客户详情页

三列布局：

| 区域 | 占比 | 内容 |
|------|------|------|
| 左列 | 35% | 聊天记录（消息气泡区分 AI/客服/客户，含翻译内容） |
| 中列 | 35% | AI 画像（CustomerAIProfile 组件，展示所有标签维度） |
| 右列 | 30% | 跟进记录（类型：电话/微信/邮件/拜访/其他，时间戳+内容） |

### 4.3 平台工单管理

管理各平台社交账号的在线状态和会话统计。

| 功能项 | 说明 |
|--------|------|
| 分组侧边栏 | 按部门分组筛选（全部/默认分组/销售部/客服部等） |
| 筛选 | 工单平台、激活码(备注)、工单状态 |
| 批量操作 | 批量选择、导出数据 |
| 工单置零 | 每日 UTC+8 凌晨自动置零，也可手动触发 |

工单状态：
- normal（正常）：账号在线运行中
- abnormal（异常）：账号出现异常
- offline（离线）：账号未登录

表格列：工单号、工单平台、激活码(备注)、工单状态、在线会话/总会话、操作

底部汇总行：在线会话合计 / 总会话合计

### 4.4 内控管理

#### 4.4.1 聊天记录审计

管理员可查看所有客服账号的完整聊天记录，用于合规审计和服务质量监控。

| 功能项 | 说明 |
|--------|------|
| 多维度筛选 | 激活码、平台、账号、时间范围、关键词 |
| 会话列表 | 展示单聊/群聊会话，含状态和优先级 |
| 消息查看 | 完整消息内容，标注 AI 生成消息和翻译内容 |

#### 4.4.2 内控报表

三行数据看板布局：

第一行（3 列）：

| 看板 | 指标 |
|------|------|
| 社交账号在线总数 | 实时在线/离线分布 |
| 对话总数 | 今日对话数，按发起方分类（客户/主动/AI），环比昨日 |
| 回复效率 | 平均首次响应时长、平均回复时长、回复率、平均对话轮次 |

第二行（2 列）：

| 看板 | 指标 |
|------|------|
| 超时未回复 | 今日超时次数，按人员条形图排行 |
| 合规统计（本周） | 敏感词触发、删除消息、删除联系人、发送名片、发送文件、异常登录 |

第三行：

| 看板 | 指标 |
|------|------|
| 数据排行 | 日报/周报/月报切换，对话总数/消息总数/主动发起/接待客户数排行 |

### 4.5 AI 员工设置

AI 员工是企业配置的虚拟销售/客服角色，可绑定到不同平台自动接待客户。

#### 4.5.1 AI 员工列表页

B 端标准表格列表，展示所有 AI 员工。

| 列 | 说明 |
|----|------|
| 员工名称 | 头像 + 名称 |
| 人设模板 | 当前使用的人设模板名称 |
| 绑定平台 | 已激活的平台图标标签 |
| 工作时间 | 开始-结束时间 |
| 启用 | 开关控件，可直接在列表中切换启用/停用 |
| 操作 | 配置（进入详情）/ 删除 |

创建 AI 员工弹窗：
- 输入员工名称
- 选择初始绑定平台（11 平台网格选择器）

#### 4.5.2 AI 员工详情/配置页

两列布局，左侧信息卡 + 右侧配置区。

左侧信息卡（固定宽度 w-72）：

| 区域 | 内容 |
|------|------|
| 头像区 | AI 员工头像 + 启用状态指示灯 |
| 名称 | 员工名称 + 人设模板标签 |
| 启用开关 | 启用/停用状态切换 |
| 已激活平台 | 平台图标标签列表 |
| 已开启功能 | 功能标签（智能销售/主动营销/客户召回/质量检测） |
| 语气预览 | 模拟问答示例，展示当前人设的回复风格 |

右侧配置区（可滚动）：

**AI 身份/人设卡片：**
- 昵称输入 + 保存按钮
- 人设模板选择（3 种模板，3 列网格）：

| 模板 | 图标 | 特点 | 标签 |
|------|------|------|------|
| 金牌销售型 | 🦁 | 高转化率，主动引导成交 | 高转化、主动引导 |
| 贴心客服型 | 🐬 | 高满意度，情绪识别安抚 | 高满意度、情绪识别 |
| 品牌专家型 | 🦊 | 专业知识输出，维护品牌调性 | 专业知识、品牌调性 |

**工作时间卡片：**
- 开始时间 / 结束时间（time picker）
- 工作日选择（周日~周六，多选按钮组）

**AI 能力卡片（2x2 网格，每项独立开关）：**

| 能力 | 说明 |
|------|------|
| 智能销售对话 | 自动识别客户意图，智能推荐产品并引导下单 |
| 主动营销触达 | 根据客户画像自动发送个性化营销消息 |
| 客户召回 | 自动识别沉默客户并发送召回消息 |
| 质量检测 | 自动检测对话质量并生成改进建议 |

> 注：AI 能力按平台独立配置，多平台时顶部有平台选项卡切换。

### 4.6 企业知识库（企业级）

知识库、话术库、AI 标签为企业级资源，所有 AI 员工共享，不按员工隔离。

#### 4.6.1 知识库管理

左右两栏布局：左侧知识库列表，右侧文档列表。

| 功能项 | 说明 |
|--------|------|
| 创建知识库 | 名称、描述、颜色标识（10 种预设色） |
| 删除知识库 | 确认后删除 |
| 文档上传 | 支持 PDF/DOCX/DOC/XLSX/XLS/CSV/TXT/MD |
| 上传向导 | 3 步流程：选择数据源 → 文本分段清洗 → 处理完成 |
| 文档启用/禁用 | 单文档级别开关 |
| 文档搜索 | 按文件名搜索 |

文档处理流水线（模拟 RAG）：
```
parsing（解析）→ chunking（分段）→ embedding（向量化）→ ready（就绪）
```

#### 4.6.2 话术库管理

> 当前状态：功能规划中，尚未实现。

预期功能：
- 管理 AI 员工的营销话术和回复模板
- 按场景分类（开场白、产品介绍、异议处理、促单话术等）
- 支持多语言版本

## 五、AI 能力体系

### 5.1 能力分层

ChatBiz 的 AI 能力分为三个层级：

| 层级 | 作用域 | 说明 |
|------|--------|------|
| 会话级 AI | 单条会话 | 翻译、回复建议、会话总结、接管模式 |
| 员工级 AI | AI 员工 × 平台 | 智能销售、主动营销、客户召回、质量检测 |
| 企业级 AI | 全组织 | 知识库、话术库、AI 标签画像体系 |

### 5.2 会话级 AI 功能

| 功能 | 触发方式 | 输出 |
|------|----------|------|
| AI 自动翻译 | 收到消息自动触发 | 将客户消息翻译为中文，发送时翻译为客户语言 |
| AI 回复建议 | 每条会话自动生成 | 3 条候选回复，一键采用 |
| AI 会话总结 | 手动或自动触发 | 当前会话的结构化摘要 |
| AI 接管模式 | 手动开启 | AI 全自动回复客户消息，无需人工干预 |

### 5.3 员工级 AI 能力

每个 AI 员工可按平台独立配置以下 4 项能力：

| 能力 | 说明 | 典型场景 |
|------|------|----------|
| 智能销售对话 | 识别客户意图，推荐产品，引导下单 | 客户询价时自动推荐匹配产品 |
| 主动营销触达 | 根据画像发送个性化营销消息 | 向高意向客户推送促销活动 |
| 客户召回 | 识别沉默客户并发送召回消息 | 7 天未互动客户自动触达 |
| 质量检测 | 检测对话质量并生成改进建议 | 发现客服回复不规范时提醒 |

### 5.4 AI 功能使用情况统计

管理中心提供 AI 功能的运营数据看板：

| 指标 | 说明 |
|------|------|
| 今日/本周接待人数 | AI 员工自动接待的客户数 |
| 回复消息数 | AI 生成并发送的消息总数 |
| AI 建议采纳率 | 客服采纳 AI 回复建议的比例 |
| 平均响应时间 | AI 首次回复的平均耗时 |
| 语言能力统计 | 按语言分布（中/英/日/韩/其他） |
| 每日趋势图 | 近 7 天 AI 使用量趋势 |

### 5.5 多语言支持

| 类别 | 支持范围 |
|------|----------|
| 界面语言 | 中文、English、日本語、한국어、Italiano、Español、Français、Deutsch、Русский、العربية |
| AI 翻译 | 自动检测客户语言，双向翻译 |
| 国家/货币 | 中国(CNY)、美国(USD)、英国(GBP)、日本(JPY)、韩国(KRW)、意大利/德国/法国/西班牙(EUR)、澳大利亚(AUD)、加拿大(CAD) |
