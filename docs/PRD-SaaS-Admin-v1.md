## 二、产品定位

SaaS 总控端是**平台运营方（我方）**使用的超级管理后台，用于管理所有企业租户的开通、计费、AI 能力下发及系统运营。

与企业管理中心的区别：

| 维度 | SaaS 总控端 | 企业管理中心 |
| --- | --- | --- |
| 使用方 | 平台运营人员（我方） | 企业管理员（客户方） |
| 管理对象 | 租户、套餐、收款、AI模板 | 员工、激活码、客户、AI配置 |
| 数据范围 | 全平台所有租户 | 当前企业内部 |

## 五、一期功能详细说明

### 5.1 概览大盘

**业务目标**：运营人员打开系统第一屏即可掌握平台全局运营状态，快速发现异常。

| 区域 | 内容 |
| --- | --- |
| 核心指标卡（4项） | 在线租户数、本月收入、AI 调用量（tokens）、7天内到期租户数 |
| 租户列表 | 最近活跃租户，含套餐/端口/到期日/收入/状态，可快速进入详情 |
| AI 用量分布 | 按能力类型展示 token 消耗占比（销售回复/翻译/画像标签/会话总结） |
| 最近支付 | 最新5条交易记录，含租户名/类型/金额/状态 |
| 套餐分布 | 基础版/专业版/企业版租户数量占比 |
| 系统健康 | API 服务、AI 服务、VPN 节点、支付回调的延迟与状态 |
| 快速操作 | 新建租户、发布公告、新增套餐、查看日志 |

---

### 5.2 租户管理

**业务目标**：管理平台上所有企业租户的生命周期，从创建到禁用，支撑日常运营。

#### 5.2.1 租户列表

| 字段 | 说明 |
| --- | --- |
| 租户名称 | 企业名称 + 唯一租户ID |
| 绑定套餐 | 当前套餐名称及周期（月/季/年） |
| 端口用量 | 已用端口数 / 套餐端口上限 |
| AI 坐席用量 | 已绑定 AI 账号数 / 套餐坐席上限 |
| 到期时间 | 到期日期，7天内标橙色预警 |
| 状态 | 正常 / 即将到期 / 已到期 / 已禁用 |
| 本月收入 | 该租户本月产生的收款金额 |
| 操作 | 查看详情 / 编辑 / 禁用 / 续费 |

支持筛选：套餐类型、状态、到期时间范围、关键词搜索

#### 5.2.2 创建/编辑租户

表单字段：
1. **基本信息**：租户名称、联系人、联系邮箱、联系手机
2. **套餐配置**：选择套餐、计费周期（月/季/年）、开始日期
3. **平台权限**：勾选开通的社交平台（WhatsApp/Telegram/Line 等）
4. **AI 坐席**：坐席数量、分配模式（固定/动态）
5. **备注**：内部备注

#### 5.2.3 租户详情页

三栏布局：
- **左侧**：租户基本信息卡 + 当前套餐 + 到期时间 + 操作按钮（续费/升级/禁用）
- **中间**：激活码用量、平台账号数、AI 坐席使用情况、最近登录记录
- **右侧**：订单历史列表（时间倒序）

#### 5.2.4 到期续费处理

- 到期前 7 天：列表标橙色、概览大盘预警卡展示
- 到期前 3 天：可触发系统通知发送到租户联系邮件
- 到期后：系统自动降级为只读模式，保留数据，等待续费后恢复

---

### 5.3 套餐管理

**业务目标**：管理平台对外销售的套餐产品，控制各套餐的功能权限和资源配额。

#### 5.3.1 套餐列表

| 字段 | 说明 |
| --- | --- |
| 套餐名称 | 基础版 / 专业版 / 企业版 |
| 价格体系 | 月付价 / 季付价（9折）/ 年付价（8折） |
| 端口上限 | 该套餐最多可添加的平台账号数 |
| AI 坐席上限 | 可绑定 AI 员工的账号数量 |
| 平台权限 | 开放的社交平台列表 |
| 功能权限 | 知识库 / 内控审计 / 数据导出等高级功能开关 |
| 在售租户数 | 当前使用该套餐的租户数量 |
| 状态 | 在售 / 下架 |

#### 5.3.2 创建/编辑套餐

配置项：
1. **基础信息**：套餐名称、描述、是否展示在官网
2. **价格配置**：月付/季付/年付价格（可单独设置折扣比例）
3. **资源配额**：端口上限、AI 坐席上限
4. **平台权限**：多选支持的社交平台
5. **功能权限**：勾选开放的高级功能（知识库/内控报表/数据导出/API接入等）
6. **套餐标签**：可标记为「推荐」「热销」「新品」

**规则**：已有租户使用中的套餐不可直接删除，只能下架（不影响存量租户）

### 5.4 支付配置

**业务目标**：配置平台收款方式，支持多种支付网关，为租户提供在线支付能力。

#### 5.4.1 支付网关管理

支持接入：

| 网关 | 适用场景 | 币种 |
| --- | --- | --- |
| Stripe | 国际信用卡、订阅扣款 | USD/EUR/GBP 等 |
| PayPal | 国际买家常用 | 多币种 |
| 支付宝国际版 | 东南亚/华人市场 | 多币种 |
| 微信支付 | 国内/出海华人 | CNY |

每个网关配置项：
- API Key / Secret Key
- Webhook 回调地址（系统自动生成）
- 支持币种
- 启用/停用开关
- 测试模式 / 生产模式切换

#### 5.4.2 收款货币配置

- 默认结算货币设置
- 汇率更新方式（实时 / 固定汇率）
- 多币种展示规则（租户付款页面显示哪些货币选项）

#### 5.4.3 支付回调监控

- 回调成功率实时展示
- 失败回调列表（支持手动重试）
- 回调日志查询

---

### 5.5 订单与账单

**业务目标**：记录和管理所有租户的购买订单与收款账单，支持财务对账和退款操作。

#### 5.5.1 订单管理

**订单类型**：
- 新购套餐
- 套餐续费
- 套餐升级（差价补款）
- 购买附加包（AI加油包/端口扩展包）

**订单列表字段**：

| 字段 | 说明 |
| --- | --- |
| 订单号 | 系统自动生成唯一编号 |
| 租户名称 | 关联租户 |
| 订单类型 | 新购/续费/升级/附加包 |
| 商品名称 | 套餐名称 + 计费周期 |
| 金额 | 实付金额 + 币种 |
| 支付方式 | Stripe / PayPal / 支付宝等 |
| 支付状态 | 待支付 / 已支付 / 已退款 / 支付失败 |
| 创建时间 | 订单创建时间 |
| 支付时间 | 实际完成支付时间 |
| 操作 | 查看详情 / 退款 / 手动核销 |

支持筛选：租户、订单类型、支付状态、时间范围、金额区间

#### 5.5.2 支付流水

独立于订单的原始收款记录（来自支付网关回调）：

| 字段 | 说明 |
| --- | --- |
| 流水号 | 支付网关返回的交易ID |
| 关联订单 | 对应的系统订单号 |
| 到账金额 | 实际到账金额（扣除网关手续费后） |
| 网关手续费 | 支付网关收取的手续费 |
| 支付网关 | Stripe / PayPal 等 |
| 到账时间 | 资金到账时间 |

#### 5.5.3 退款操作

退款流程：
1. 运营人员在订单详情点击「申请退款」
2. 填写退款原因 + 退款金额（支持部分退款）
3. 系统向支付网关发起退款请求
4. 退款状态同步回订单记录
5. 租户订阅状态相应调整（全额退款则关闭订阅）

#### 5.5.4 对账报表

- 按日/周/月汇总收款金额
- 按套餐类型分组统计
- 导出 Excel（供财务使用）

### 5.6 AI 管理中心（已完成）

**业务目标**：平台统一维护 AI 内容模板，下发给所有租户，租户可在权限范围内自定义。

**已实现功能**：

| 子模块 | 说明 |
| --- | --- |
| 角色人设 | 6种人设风格卡片，支持激活、启用开关、提示词抽屉编辑 |
| 模型配置 | 6个AI场景独立配置大语言模型（GPT-4o/Claude/DeepSeek等） |
| 意图库 | 单聊/群聊意图，场景-话术多行管理，精准锁定 vs 随人设调整 |
| 策略库 | 单聊/群聊策略，触发条件（用户行为/时间/订单状态）+ 执行话术 |
| 话术库 | 欢迎语话术，按触发关键词分组管理 |
| 关键词回复 | 单聊/群聊关键词，支持批量导入 |

**下发规则**：平台下发的模板为默认值，租户已有自定义版本不会被覆盖。

**业务目标**：管理运营团队内部账号的访问权限，防止越权操作。

#### 5.7.1 运营账号管理

| 字段 | 说明 |
| --- | --- |
| 账号名 | 登录用户名（全局唯一） |
| 姓名 | 真实姓名 |
| 角色 | 超级管理员 / 运营人员 / 财务人员 |
| 状态 | 启用 / 停用 |
| 最近登录 | 最后一次登录时间 + IP |

#### 5.7.2 角色权限配置

| 角色 | 可访问模块 |
| --- | --- |
| 超级管理员 | 全部模块 |
| 运营人员 | 概览大盘、租户管理、套餐管理、AI管理中心 |
| 财务人员 | 概览大盘（只读）、订单与账单、支付流水 |

## 六、二期功能规划

### 6.1 附加包管理

- AI 加油包：定义不同规格的 token 充值包（如 100万/500万/1000万 tokens）及定价
- 端口扩展包：在套餐基础上追加端口数量的独立购买项
- 附加包列表、上下架、购买记录追踪

### 6.2 IP 管理

- 平台代理 IP 池维护（添加/删除/批量导入）
- IP 健康检测（延迟/可用性定时检测）
- IP 分配策略（按租户套餐等级优先分配优质 IP）
- IP 使用率统计

### 6.3 AI 用量统计

- 按租户维度：各AI能力的 token 消耗明细
- 趋势图：日/周/月消耗走势
- 超限预警：接近配额上限时告警
- 成本分析：按模型厂商统计 API 调用费用

### 6.4 系统配置

- 全局公告发布（所有租户登录时展示）
- 功能灰度开关（按租户/套餐灰度发布新功能）
- 翻译服务 API Key 配置（谷歌/DeepL/百度/有道）
- 各 AI 模型 API Key 配置与余额监控
- 系统限流参数配置

### 6.5 日志管理

- 操作日志：运营账号的所有操作记录（谁/时间/操作内容/结果）
- 租户登录日志：租户管理员登录记录 + 异常登录告警
- 支付回调日志：全量支付回调记录，支持回放和排查
- 日志搜索与导出

## 七、租户自助订阅流程（配合支付模块）

```
租户访问官网定价页
    │
    ▼
选择套餐 + 计费周期
    │
    ▼
填写企业信息（首次注册）
    │
    ▼
选择支付方式（Stripe / PayPal / 支付宝）
    │
    ▼
完成支付
    │
    ├── 支付成功 → 系统自动创建租户 + 发送激活邮件 → 进入管理中心
    │
    └── 支付失败 → 提示重试，订单保留 30 分钟
```

**续费流程**（到期前自动提醒）：
- 到期前 7 天：邮件提醒 + 管理中心 Banner 提示
- 到期前 3 天：再次邮件提醒
- 到期后：只读模式，续费后立即恢复，有效期从原到期日顺延
